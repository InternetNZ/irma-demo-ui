Wine shop

During a Wine Shop, we check if the user has an attribute of age limit as a proof that the user is over 18. The user can also opt if he wants to buy as a guest or to sign a subscription. As a guest buyer, an address will be asked, as a subscriber, name and address will be asked. On both cases, the user can fill manually or fill with IRMA attributes.

Age limits, address and name may be issued at simple issuer page.
